## 🔄 预约业务流程

```
开始
  │
  ├─→ 用户浏览剧本 (ScriptController.page)
  │   └─→ 返回 List<Script>
  │
  ├─→ 选择门店 (StoreController.page)
  │   └─→ 返回 List<Store>
  │
  ├─→ 查看排期 (ScriptScheduleController.querySchedules)
  │   └─→ 返回 List<ScriptSchedule>
  │   └─→ 检查库存: currentPlayers < maxPlayers?
  │
  ├─→ 输入预约信息
  │   ├─ playerCount
  │   ├─ reservationTime
  │   ├─ contactName
  │   └─ contactPhone
  │
  ├─→ 选择优惠券/VIP (可选)
  │   ├─ CouponService.getAvailableCoupons()
  │   ├─ VipService.getVipDiscount()
  │   └─ 计算最终价格
  │
  ├─→ 创建预约 (ReservationController.create)
  │   │
  │   └─→ ReservationServiceImpl.create()
  │       │
  │       ├─ 验证房间可用性
  │       ├─ 验证优惠券有效性
  │       ├─ 计算VIP折扣
  │       ├─ 计算最终价格
  │       ├─ 插入 Reservation 表
  │       │   └─ status = 待支付
  │       │   └─ payStatus = 未支付
  │       ├─ 更新 ScriptSchedule.currentPlayers++
  │       ├─ 分配核销码
  │       └─ 返回预约对象
  │
  ├─→ 显示订单信息
  │   └─ orderNo, totalPrice, actualAmount
  │
  ├─→ 支付宝支付 (PaymentController.pay)
  │   │
  │   └─→ PaymentServiceImpl.pay()
  │       │
  │       ├─ 调用支付宝API
  │       ├─ 生成支付二维码/链接
  │       └─ 返回支付信息
  │
  ├─→ 支付回调处理 (PaymentController.notifyAlipay)
  │   │
  │   └─→ PaymentServiceImpl.handleCallback()
  │       │
  │       ├─ 验证签名
  │       ├─ 更新 Reservation.payStatus = 已支付
  │       ├─ 更新 Reservation.payTime
  │       ├─ 生成核销码
  │       ├─ 发送支付成功通知 (WebSocket)
  │       └─ 积分奖励 (UserPointsService.addPoints)
  │
  ├─→ 预约即将开始
  │   │
  │   └─→ ReservationTask.checkUpcomingReservations()
  │       │
  │       └─ 发送提醒通知 (NotificationService)
  │
  ├─→ 用户到店
  │   │
  │   └─→ ReservationController.checkIn(checkInCode)
  │       │
  │       └─→ ReservationServiceImpl.checkIn()
  │           │
  │           ├─ 验证核销码
  │           ├─ 更新 checkInStatus = 已核销
  │           ├─ 更新 checkInTime
  │           └─ 解锁评价功能
  │
  ├─→ 游戏完成后评价
  │   │
  │   └─→ ReviewController.submit()
  │       │
  │       └─→ ReviewServiceImpl.submit()
  │           │
  │           ├─ 保存 Review 表
  │           ├─ 更新评分 (Store, Script, StoreEmployee)
  │           ├─ 奖励积分和优惠券
  │           └─ 发送通知
  │
  └─→ 预约完成

可能的中途事件:
  │
  ├─→ 用户取消预约 (ReservationController.cancel)
  │   ├─ 检查是否可取消 (status, time)
  │   ├─ 退款处理
  │   ├─ 回收优惠券
  │   ├─ 减少 ScriptSchedule.currentPlayers
  │   └─ 积分扣除
  │
  ├─→ 用户改期 (ReservationController.reschedule)
  │   ├─ 检查新时段可用性
  │   ├─ 更新 reservationTime
  │   ├─ 更新 scheduleId
  │   └─ 发送改期通知
  │
  └─→ 支付超时 (ReservationTask.checkUnpaidReservations)
      └─ 自动取消并回收库存
```
