## 一、项目概述

### 1.1 项目背景

剧本杀作为近年来快速发展的娱乐形式，市场规模持续扩大。本系统旨在为剧本杀门店提供一站式的预约管理解决方案，同时为玩家提供便捷的预约和社交体验。

### 1.2 项目定位

- **用户端**：面向C端玩家，提供剧本浏览、门店查找、在线预约、拼团组队等功能
- **管理端**：面向B端商家，提供门店管理、预约管理、数据统计、营销工具等功能

### 1.3 技术栈

| 层级 | 技术选型 |
|------|----------|
| 后端框架 | Spring Boot 3.x |
| 持久层 | MyBatis Plus |
| 缓存 | Redis |
| 数据库 | MySQL 8.0 |
| 前端框架 | Vue 3 |
| UI组件库 | Element Plus |
| 构建工具 | Vite |
| 状态管理 | Pinia |
| 实时通信 | WebSocket |
