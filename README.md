# Murder Boot - 剧本杀预约与门店管理系统

> Spring Boot 单体应用版本，整合了原有的微服务架构

---

## 📖 项目简介

Murder Boot 是由 Spring Cloud 微服务架构迁移而来的 Spring Boot 单体应用，保持了相同的功能和使用方式。

**技术栈**：
- 后端：Spring Boot 3.x + MyBatis Plus + Redis + MySQL
- 前端：Vue 3 + Element Plus + Vite
- 容器：Docker (Redis)

---

## 🚀 快速开始

### 1️⃣ 启动后端

```bash
# 启动 Redis
docker start murder-redis

# 在 IDEA 中运行 MurderBootApplication.java
# 等待启动成功
```

### 2️⃣ 启动用户端前端

```bash
cd murder-boot/frontend/user
npm run dev
```

**访问**：http://localhost:3001

### 3️⃣ 启动管理端前端

```bash
cd murder-boot/frontend/admin
npm run dev
```

**访问**：http://localhost:3000

## 🔧 开发说明

### 开发模式（推荐）
- 前端独立运行（3000/3001 端口）
- 后端独立运行（8001 端口）
- 通过 Vite 代理访问 API
- 支持热重载

### 生产模式
- 前端构建后部署到后端 static 目录
- 或前端独立部署到 Nginx
- 后端打包成 jar 文件运行

## 🐛 常见问题

### 端口被占用
```bash
netstat -ano | findstr :3000
taskkill /PID <进程ID> /F
```

### API 请求失败
1. 检查后端是否运行在 8001
2. 检查 Redis 是否启动
3. 查看浏览器控制台错误

### 登录后 401 错误
1. 清除浏览器缓存
2. 重启后端应用
3. 重新登录
